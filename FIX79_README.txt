FIX79 出貨不再卡在整理母版

1. 出貨確認送出改成先立即顯示本機出貨預覽，避免卡在「整理預覽中」。
2. 背景再載入可扣來源與倉庫位置；背景逾時也不鎖住按鈕。
3. 關閉舊版加入商品時自動打 /api/ship-preview 的無逾時流程。
4. 確認扣除仍走 /api/ship，保留總單 → 訂單 → 庫存扣除順序。
5. 保留 073 / 06 / 006 前導 0、支數件數規則、材積與重量計算、不顯示長度計算。
6. 後端出貨倉庫位置查詢改成唯讀，不再在出貨預覽流程中補格寫入，避免資料庫鎖等待。
7. 版本號更新到 fix79-ship-no-stuck-master，避免舊快取。
