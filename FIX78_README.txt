FIX78 出貨錯誤原因檢查與全功能穩定修正版

已修正：
1. 出貨確認送出卡住：前端改為 FIX78 最終出貨流程，按下後一定會解除按鈕鎖，失敗會顯示明確錯誤，不會停在整理中。
2. 出貨預覽：仍先顯示出貨預覽，再按確認扣除才真正扣除。
3. 扣除順序：維持總單 → 訂單 → 庫存。
4. 0xx 高度：073 / 063 / 006 / 06 顯示與材積計算都保留，不再刪 0。
5. 支數件數：右側 xxxx / xxx / xx / x 未帶 x數量時一律算 1 件；有 xN 算 N 件。
6. 長度計算：出貨預覽不顯示長度計算，只保留件數、材積、重量、可扣來源、倉庫位置。
7. 舊資料相容：如果舊資料曾把 073 存成 73，出貨扣除會用同尺寸相容比對，避免預覽有資料但扣不到。
8. 部分出貨相容：出貨時若只修改本次件數/支數，也能扣同尺寸商品，不會因右側支數文字不同而扣不到。
9. 已選商品區：保留一鍵清空，移除缺貨卡片與上移/下移/移除顯示。
10. 快取版本：base / pwa / service worker / manifest 已更新到 FIX78。

保留既有功能：OCR、庫存、訂單、總單、出貨、倉庫圖、客戶資料、今日異動、批量材質/刪除、合併提示、長按倉庫格返回上一步等功能均保留。
